API de Preguntas de Trivia Aleatorias

Conectar con la API de Open Trivia Database

Open Trivia Database (OpenTDB) es una API gratuita que proporciona preguntas de trivia en formato JSON. Puedes personalizar la cantidad de preguntas, la categoría, la dificultad y el tipo de respuesta. Para acceder a la API, utilizamos el siguiente endpoint: https://opentdb.com/api.php. Un ejemplo de solicitud sería: https://opentdb.com/api.php?amount=10, donde el parámetro amount=10 indica que queremos obtener 10 preguntas aleatorias.

Para mostrar estas preguntas en una página web, en el caso del endpoint https://opentdb.com/api.php?amount=10 retorna un .json con un arreglo con el type, difficulty, category, question, correct_answer, y las incorrect_answers como elementos de cada pregunta en el arreglo. El Javascript se vería así dentro de una función:

const apiUrl = "https://opentdb.com/api.php?amount=10"; const respuesta = await fetch(apiUrl); const rJSON = await respuesta.json(); const trivia = document.getElementById("Respuestas"); rJSON.results.forEach((item, index) => { const questionText = document.createTextNode(`${item.question}: `); const answerText = document.createTextNode(`${item.correct_answer}. `); trivia.appendChild(questionText); trivia.appendChild(document.createElement("br")); trivia.appendChild(answerText); trivia.appendChild(document.createElement("br")); trivia.appendChild(document.createElement("br"));
